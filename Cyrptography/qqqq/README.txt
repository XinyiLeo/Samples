	In this program I developed two layers: Shift Rows layer, and Subbytes layer
	
	To run the program, first type : make
	then there are four choices:
	
	./hw2 1			encrypt with Shift Rows layer, user need type the plaint text				

	./hw2 2			encrypt with Shift Rows layer, user need type the plaint text	

	./hw2 1 default		encrypt with Shift Rows layer, use the default plain text

	./hw2 2 default		encrypt with SubBytes layer, use the default plain text
